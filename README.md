# WeatherApp

WeatherApp est une application météo cross-platform construite avec Expo et React Native. Elle offre des prévisions météorologiques en temps réel avec localisation automatique et une interface utilisateur intuitive sur iOS, Android et Web.

## 🚀 Démarrage rapide

1. Installer les dépendances

   ```bash
   npm install
   ```

2. Lancer l'application

   ```bash
   npx expo start
   ```

3. Ouvrir sur :
   - [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
   - [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
   - [Expo Go](https://expo.dev/go)
   - Web browser

## 📁 Structure du projet

Le projet utilise [file-based routing](https://docs.expo.dev/router/introduction/) d'Expo Router.

- `app/` - Pages et écrans de l'application
- `components/` - Composants React réutilisables
- `hooks/` - Hooks personnalisés
- `services/` - Services (localisation, météo)
- `types/` - Définitions TypeScript
- `constants/` - Constantes et thème
